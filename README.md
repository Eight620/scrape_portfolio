# 書籍データ自動収集 & Excel出力ツール

## 概要
PythonのSeleniumを使用し、ECサイト（デモ用サイト）から特定カテゴリの書籍情報を収集するツールです。
ブラウザ操作の自動化により、「Travel」カテゴリへの遷移を行い、書籍のタイトルと価格を抽出・整形してExcelファイルとして保存します。

## 機能・工夫した点

### 1. 正確なデータ抽出
Webページ上の表示テキスト（`.text`）は、タイトルが長い場合に省略（`...`）されることがあります。
本ツールではHTMLの `title` 属性から値を取得することで、省略されていない**完全な正式名称**を収集するように実装しています。

### 2. データ整形（Data Cleaning）
収集した価格データ（例: `£54.23`）をそのまま文字列として保存するのではなく、通貨記号を除去し**数値型（float）に変換**しています。
これにより、出力されたExcelファイルですぐに計算や並び替えが可能になります。

### 3. カテゴリへの自動遷移
トップページから特定のリンク（Travel）を探索し、クリック操作でページ遷移を行う動的なスクレイピングを実装しています。

### 4. 安定した動作
`implicitly_wait`（暗黙的な待機）を設定し、ページの読み込み遅延によるエラーを防ぐ堅牢な設計にしています。

## 使用技術
- **言語**: Python 3.x
- **ブラウザ操作**: Selenium WebDriver
- **データ処理**: Pandas
- **ファイル出力**: Excel (.xlsx)
